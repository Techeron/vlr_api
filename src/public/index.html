What is this?

This website is an API for the vlr.gg website.
It is a work in progress and is not complete.
It is not affiliated with vlr.gg in any way.
It is a personal project that I am working on to learn more about APIs and web development.
If you have any questions or suggestions, feel free to contact me on discord at BubblySpore#4126

How Does this work?

/api/:type/:id

For Example: /api/player/1337 will show details about Naga
Types: 

event | events | player | team

WIP Types: 
players | teams | news

Id: the id of the player, team or event from the vlr.gg site
Plural types will return an array of objects, while singular types will return a single object.
Plural types do not need an id
All requests made are get requests, and for now there is no caching system so all data is pulled live from the site every time you make a request. This is not ideal and will be changed in the future.
